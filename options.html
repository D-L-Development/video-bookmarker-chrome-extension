How To Use:

Shortcut

Description
CTRL + B Create a new bookmark. Will pause video until finished, and then start playback again.
CTRL + ; Copy bookmarks to clipboard and pretty print to the console. Formats the copied bookmarks to be able to paste in Google Sheets.
CTRL + , Set a new session name.
